Insurance Policy - Agentic AI System

📊 시스템 전체 구조

ISPL 시스템의 3계층 아키텍처를 보여줍니다. Frontend (React/Next.js), Backend (FastAPI + LangGraph), 그리고 외부 서비스 (OpenAI, PostgreSQL, 파일 시스템)로 구성됩니다.

🤖 LangGraph Multi-Agent 상세

5개의 전문화된 Agent (Router, Search, Answer, Processing, Management)의 워크플로우와 상호작용을 상세히 보여줍니다.

⚙️ PDF 전처리 하이브리드 파이프라인

Path 1 (PyMuPDF4LLM 직접 추출)과 Path 2 (GPT-4 Vision 멀티모달)를 병렬로 실행하고 결과를 병합하는 고품질 전처리 과정을 보여줍니다.

🔍 검색 및 답변 생성 플로우

사용자 질의부터 최종 답변까지의 전체 프로세스를 단계별로 보여줍니다. 질의 전처리, 하이브리드 검색, GPT-4 답변 생성, 할루시네이션 검증 과정을 포함합니다.

🗄️ 데이터베이스 스키마

PostgreSQL 데이터베이스의 테이블 구조와 관계를 보여줍니다. users, documents, document_chunks (벡터 임베딩), processing_logs, search_logs 테이블로 구성됩니다.

🛠️ 기술 스택

Frontend (Next.js, React, TypeScript), Backend (FastAPI, Python, LangGraph), AI/ML (OpenAI GPT-4, Embeddings), Database (PostgreSQL + pgvector) 등 프로젝트에서 사용하는 모든 기술과 라이브러리를 정리했습니다.
